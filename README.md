# oe-lifecycle-manager
oe-lifecycle-manager is an enterprise-grade operating system lifecycle management platform designed for openEuler environments. It provides offline in-place upgrade, risk assessment, compatibility analysis, rollback protection, and lifecycle governance capabilities for production systems.

## Current phase

The project is currently in the architecture design phase. See [docs/architecture-design.md](docs/architecture-design.md) for the overall architecture, module design, class model, database schema, workflow state machine, plugin mechanism, API design, risk model, and testing strategy.
